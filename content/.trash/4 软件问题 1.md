## 5.  Flatpak 
### 5.1.  文件目录
`~/.var/app/` 目录是用于存储使用 [Flatpak](https://flatpak.org/) 技术安装的应用程序的特定用户数据和配置文件的目录。

### 5.2.  设置软件访问权限
#### 5.2.1.  允许访问整个文件系统
为某个 Flatpak 应用程序打开对整个文件系统的访问权限：

```bash
flatpak override --filesystem=host com.example.AppName
```
将 `com.example.AppName` 替换为你安装的应用程序的实际名称。

#### 5.2.2.  允许访问特定目录
如果你只想让应用程序访问特定目录，例如 `/mnt/data`，你可以执行以下命令：

```bash
flatpak override --filesystem=/mnt/data com.example.AppName
```

#### 5.2.3.  允许访问主目录
如果你只想让应用访问主目录，可以运行以下命令：

```bash
flatpak override --filesystem=home com.example.AppName
```
### 5.3.  检查当前应用权限
你可以查看某个应用程序当前的权限，执行以下命令：

```bash
flatpak info --show-permissions com.example.AppName
```

## 6.  卸载软件(以 Firefox 为例)

### 6.1.  彻底卸载 firefox
1. 删除 firefox 及其配置文件
```sh
sudo apt-get remove --purge firefox
```

> `--purge` 选项会删除 Firefox 及其配置文件。

2. 删除残留的配置文件和缓存
```sh
rm -rf ~/.mozilla/firefox/ # 删除配置文件
rm -rf ~/.cache/mozilla/firefox/ # 删除缓存文件
```

3. 检查并删除可能的剩余包
```sh
sudo apt-get autoremove
```

### 6.2.  安装 firefox

```sh
sudo apt install firefox
```

## 7.  修改软件默认启动主题(以 zotero 为例)
如需要修改 zotero 的默认启动主题为 light
1. 进入 zotero 的主目录 `cd /opt/zotero`
2. 编辑 `zotero.desktop`.
3. 原本的字段为`Exec=bash -c "$(dirname $(realpath $(echo %k | sed -e 's/^file:\/\///')))/zotero -url %U"`
   修改为 `Exec=bash -c "$(dirname $(realpath $(echo %k | sed -e 's/^file:\/\///')))/zotero -url %U"`


## 8.  应用的配置文件保存的位置
### 8.1.  **`~/.config/`** 目录
大部分桌面应用的配置文件都保存在此。例如，如果你的 AppImage 应用是 `ExampleApp`，那么它的配置可能位于 `~/.config/ExampleApp`。
```bash
     rm -rf ~/.config/ExampleApp
```

### 8.2.  **`~/.local/share/`** 目录
有些 AppImage 应用的数据文件可能保存在此。
```bash
     rm -rf ~/.local/share/ExampleApp
```

### 8.3.  **`~/.cache/`** 目录
如果 AppImage 应用缓存了一些数据，可能会存储在此目录中。
```bash
     rm -rf ~/.cache/ExampleApp
```

### 8.4.  `~` 目录
也有可能在 `~` 目录 下的隐藏文件夹里

## 图像压缩
使用 `ImageMagick`
压缩 `jpg`:
```bash
magick input.jpg -quality 85 output.jpg

magick *.jpg -quality 85 output_%03d.jpg # 批量

magick *.jpg -quality 85 *.jpg # 覆盖
```

压缩 `png`:
```bash
magick input.png -strip output.png # 去除元数据

magick *.png -compression-level 9 *.png # 设置压缩级别
```
