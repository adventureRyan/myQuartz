## conda 创建
因为空间不足,所以只能以这种方式创建你的虚拟环境了
```
conda create --prefix /home/s23/s23hjy/workspace/conda_env/env_name python=3.8
```
> 把 env_name 换成你需要创建的环境名

下面这个是我刚刚创建的环境
![500](attachment__服务器使用/file-20241104153045440.png)

**激活虚拟环境**时,使用 
```
conda activate /home/s23/s23hjy/workspace/conda_env/test_name
```

> 无法对 conda 的环境进行重命名,所以不得不使用这个方法
