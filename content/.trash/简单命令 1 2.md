## 连接机器人
使用 ssh 连接机器人系统
```bash
ssh -X lemon@lemon-NUC8i3BEH.local
```
将系统文件挂载到本地 Linux 文件夹中
```bash
sshfs lemon@192.168.31.15:/home ~/robanrobot
```

## 常见命令  
```bash
rosnode list  # 查看所有活跃节点

# 如果摄像头打不开执行下面的命令
source ~/robot_ros_application/catkin_ws/devel/setup.bash 
rosrun web_video_server web_video_server
```

## 机器人节点重启
```bash
source ~/robot_ros_application/scripts/start.sh
```
## 部分程序
### 执行机器人走路程序
```bash
cd ~/robot_ros_application/catkin_ws/
source devel/setup.bash
rosrun gait_command key_tele.py
```

### 案例程序
